# evals/

The evaluation dataset: five manually reviewed companies, each with a
gold-standard company profile, verified signals, suitable target roles,
reasonable pain hypotheses, unacceptable claims, and one strong/one weak
example message. Built starting Chapter 10 (Integrating and Evaluating the
MVP, specifically 10.5 "Building a Beginner Evaluation Dataset"), once
research and evidence rules exist to review against. Empty as of the current
Chapter 2 checkpoint.
