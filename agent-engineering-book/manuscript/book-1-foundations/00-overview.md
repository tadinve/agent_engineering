# Book 1 — Agent Engineering Foundations
### Building the Claude SDR Lab MVP

This section establishes the essential building blocks of Agent Engineering through a bounded, inspectable project. Readers construct the first version of the Claude SDR Lab: an evidence-backed system that researches companies, identifies relevant signals, develops business hypotheses, prepares outreach and pauses for human approval. The focus is on understanding each component rather than maximizing autonomy. By the end of the section, readers will have a complete agentic workflow running locally, supported by source control, structured data contracts, test cases and a reference implementation.
