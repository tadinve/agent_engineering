# Book 2 — Advanced Agent Architectures
### Knowledge, Memory, Planning and Collaboration

Book 2 expands the bounded MVP into a persistent and adaptive agent system. Readers learn how to engineer context, retrieve internal knowledge, preserve experience, resume long-running work and coordinate more sophisticated reasoning. Memory, RAG, state and artifacts are treated as distinct architectural concepts rather than being collapsed into a single feature. Planning and loops are introduced with explicit budgets and termination conditions, while MCP and agent-to-agent communication extend the system beyond its local repository.
